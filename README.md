# E-COMMERCE-SALES-DASHBOARD

# 🛍️ E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview

The **E-Commerce Sales Dashboard** is an interactive Business Intelligence project developed using **Power BI** to analyze online retail sales performance across categories, states, and payment modes. The dashboard provides valuable insights into revenue, profitability, order volume, and customer purchasing behavior through dynamic visualizations and KPI metrics.

This project demonstrates skills in data cleaning, data modeling, DAX, and dashboard development to support data-driven business decisions.

---

# 🎯 Objectives

- Analyze overall e-commerce sales and profit performance.
- Monitor key business KPIs.
- Identify top-performing product categories and sub-categories.
- Compare sales performance across states and cities.
- Analyze customer payment preferences.
- Build an interactive dashboard for business reporting.

---

# 📂 Dataset Information

The project uses two related transactional datasets containing order and sales details.

### Orders.csv includes:

- Order ID
- Order Date
- Customer Name
- State
- City

### Details.csv includes:

- Order ID
- Amount
- Profit
- Quantity
- Category
- Sub-Category
- Payment Mode

**Categories:** Electronics, Furniture, Clothing
**Sub-Categories:** Phones, Printers, Electronic Games, Chairs, Tables, Bookcases, Furnishings, Shirt, T-shirt, Trousers, Skirt, Saree, Kurti, Leggings, Stole, Hankerchief, Accessories
**Payment Modes:** COD, Credit Card, Debit Card, EMI, UPI
**Records:** ~500 orders, ~1500 order line items
**Time Period:** January 2018 – December 2018
**Geography:** Multiple Indian states and cities

**Dataset Format:** CSV (.csv)

# 🔄 Project Workflow

1. Import the Orders.csv and Details.csv datasets into Power BI.
2. Clean and transform data using Power Query.
3. Create relationships between Orders and Details tables using Order ID.
4. Develop DAX measures for KPI calculations.
5. Design interactive dashboard visuals.
6. Analyze sales, profit, and payment trends.
7. Generate business insights and recommendations.

# ▶️ How to Run the Project

1. Clone this repository.

```bash
git clone https://github.com/bhaktig0210/Ecommerce-Sales-Dashboard.git
```

2. Open **Power BI Desktop**.

3. Open the `Ecommerce_sales_Report_DASHBOARD.pbix` file.

4. If prompted, reconnect the datasets (`Orders.csv` and `Details.csv`).

5. Refresh the data.

6. Explore the dashboard using interactive filters and slicers.
